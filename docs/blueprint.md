# **App Name**: CampusConnect

## Core Features:

- User Registration & Role Management: Handles teacher and student registration, class code generation (teacher), and college ID verification (student).
- Teacher Class Management: Teachers can create class codes and accept student registration requests.
- Attendance Tracking: Tracks student attendance and calculates the attendance percentage for each student.
- Automated Notifications: Sends notifications to students whose attendance is below 75%.
- Assignment Management: Enables teachers to send assignments with deadlines and mode (online/offline), and students to submit them.
- Notification System: Allows teachers to send announcements to students.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) for a calm and trustworthy environment.
- Background color: Light gray (#F0F8FF), a very pale cyan, providing a neutral backdrop.
- Accent color: Soft green (#98FB98), to suggest growth, learning, and positive actions like successful submissions.
- Headline font: 'Poppins', sans-serif, giving a precise and fashionable feel. For body text: 'PT Sans' for readability.
- Use clear and concise icons for navigation and actions.
- Maintain a clean and organized layout for easy navigation.
- Use subtle animations for feedback on actions and status updates.
